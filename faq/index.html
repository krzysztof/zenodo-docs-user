Frequently Asked Questions

General

Can I change the file in the already published record?

Once the DOI has been minted, it is not possible to change the files behind it. Doing so would be against a fundamental idea behind the DOI, which is the persistence of the content behind the DOI. If you have just made a new deposit and made a small typo in the text, contact us and we can replace it for you.

Technical

How can I edit the metadata of the published record?

Go to your record page and click the orange Edit button in the top-right corner. You will be taken to a form similar to the upload form, where you will be able to edit almost all of the record's metadata. Once you are done, click "Save" and then "Publish".

Policies

How much free storage can I get?

We support up to 50GB of free storage. If you need more than that, please contact us directly and we might be able to increase it depending on your specific case.

Can I pay for extra storage?

Sadly we do not accept money for extra storage.

GitHub

My organizational repository does not show up on the GitHub list.

Please refresh it - click the Sync Now button.

I made a GitHub release, but it did not end up on Zenodo.

Make sure it was enabled before the time of the publishing.

Is it possible to archive a GitHub repository, before it was enabled on Zenodo?

Only the repositories which were enabled at the time of the publishing, will be archived automatically. If you want, you can always download a ZIP from GitHub and upload it to Zenodo using our web interface.
